# 🌊 สวัสดีวันสงกรานต์

หน้าเว็บนี้ทำขึ้นเพื่ออวยพรวันสงกรานต์พร้อม animation หยดน้ำ ดอกไม้ลอย และเอฟเฟกต์สาดน้ำแบบ interactive

## ✨ Features
- 💧 หยดน้ำตกแบบ random สีสันสดใส
- 🌸 ดอกไม้ลอยขึ้นแบบ ambient
- 🌊 คลื่นน้ำ animate ด้านล่าง
- 💦 คลิกที่ไหนก็ได้บนหน้าจอเพื่อสาดน้ำ
- 🎊 ปุ่มสาดน้ำพร้อมกัน 15 จุด
- นับจำนวนครั้งที่สาดน้ำ

## 🚀 วิธีใช้งาน

เปิดไฟล์ `index.html` ในเบราว์เซอร์ได้เลยไม่ต้องติดตั้งอะไรเพิ่ม

## 🎨 Theme

**"สายน้ำแห่งความสุข"** — พื้นหลังไล่เฉดสีน้ำเงินลึกถึงเขียวมรกตสื่อถึง น้ำ, ท้องฟ้า และ ความสดชื่น
